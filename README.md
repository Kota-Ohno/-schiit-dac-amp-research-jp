# Schiit ヘッドホン／IEM向け DAC・アンプ調査

Schiit Audioのヘッドホン／イヤホン用途向けDAC、ヘッドホンアンプ、DAC/アンプを、公式仕様・独立測定・専門レビュー・ユーザー報告から整理した日本語データセットです。

調査基準日: **2026-07-18**

## 収録ファイル

| ファイル | 内容 |
|---|---|
| `data/Schiit_DAC_Amp_Research_2026-07-18.json` | 全製品、レビュー傾向、反対意見、確信度、旧世代注意、出典URLを含む構造化データ |
| `data/Schiit_DAC_Amp_Current_2026-07-18.csv` | 現行17製品のフラットな一覧 |
| `spreadsheet/Schiit_DAC_Amp_Research_2026-07-18.xlsx` | 結論、現行一覧、公式仕様、レビュー傾向、旧世代注意、出典、評価基準の7シート |

## 調査方針

- 公式仕様と主観的レビューを分離
- レビューは共通傾向と反対意見を併記
- 「レビュー確信度」は好評度ではなく、独立測定・直接比較・情報量・発売後経過の成熟度
- 旧世代のレビューを現行世代へ流用しない
- IEM、平面駆動、高インピーダンス、価格性能を1〜5の編集評価で整理

## 注意

- 価格は米国公式の新品価格で、税・送料・日本向け電源対策を含みません。
- Schiitの公式電圧選択肢は115V／230Vで、日本100Vでの保証・定格性能は明記されていません。
- DACの音質評価の多くは非ブラインド試聴に基づき、高価格が比例した可聴改善を保証するものではありません。
- 製品情報・販売状態・価格は更新される可能性があります。
